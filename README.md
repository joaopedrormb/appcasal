# Nossa Casa

PWA mobile-first para organização do casal: finanças, remédios e treinos. Stack: React 18, Vite 5, Firebase 11, Zustand, CSS Modules e `vite-plugin-pwa`.

## Requisitos

- Node.js 18+

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
